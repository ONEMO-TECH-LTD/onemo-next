# Public Assets

Static files served by Next.js at runtime. Everything here is accessible via URL at the root path.

Only **active, in-use** files belong here. Reserved copies and unused alternatives live in `asset-library/`.

| Folder | Purpose |
|---|---|
| `assets/shapes/` | 3D models (GLB) |
| `assets/materials/ultrasuede/` | Active PBR texture maps (GSG SuedeBlack 4K) |
| `assets/env/` | HDRI environment maps for studio lighting |
| `assets/` | Test artwork and other runtime images |
